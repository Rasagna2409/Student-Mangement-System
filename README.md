# 📘 Student Management System (Flask)

A simple yet professional **Student Management System web application** built using **Flask, SQLite, HTML, CSS, and Chart.js**.
It allows you to manage students and courses with a clean dashboard and visual analytics.

# 🚀 Features

* ➕ Add new students
* 📋 View student list
* ❌ Delete students
* 📊 Dashboard with Chart.js visualization
* 📚 Course management
* 🗄️ SQLite database integration
* 🎨 Simple and responsive UI

# 🛠️ Tech Stack

* **Backend:** Flask (Python)
* **Database:** SQLite / SQLAlchemy
* **Frontend:** HTML, CSS
* **Charts:** Chart.js
* **Templating:** Jinja2

# ⚙️ Installation & Setup

## 1️⃣ Clone the repository
git clone https://github.com/your-username/student-management-system.git

## 2️⃣ Move into project directory
cd student-management-system

## 3️⃣ Install dependencies
pip install flask flask_sqlalchemy

## 4️⃣ Run the application
python app.py

## 5️⃣ Open in browser
http://127.0.0.1:5000/dashboard
